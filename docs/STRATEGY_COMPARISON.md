# 전략 비교: DeFi 델타뉴트럴 LP vs HL BTC EMA 매매

> **작성일**: 2026-07-16 · 동일 기간(2025-07-16~2026-07-16, 365일) · 동일 데이터(Binance 가격, Hyperliquid 실측 펀딩) · 수수료/슬리피지 반영
> 기간 특성: **하락장** (BTC -46%, ETH -43%) — 해석 시 유의

## 1. 성과 비교 (백테스트)

| 지표 | 델타뉴트럴 LP (m=3, 4풀) | BTC EMA20/60 **1h** (quant 계획서 MVP) | BTC EMA20/60 **4h** |

조건: EMA 전략은 계획서 스펙 그대로 — ATR(14)×2 SL / ×3 TP (R:R 1.5), 리스크 1%/거래, 3x 캡, 크로스 시 반대 진입, 테이커 수수료+슬리피지 0.065%/사이드, 실측 펀딩 반영(기여도 ~0%).

## 2. 수익의 원천이 다르다 (핵심)

| | 델타뉴트럴 LP | EMA 추세추종 |
|---|---|---|
| 수익 원천 | 유동성 판매 대가 (수수료) + 펀딩 캐리 | 추세 예측 알파 |
| 수익 조건 | 시장이 **거래되기만 하면** (변동성=수익) | **추세가 지속**되어야 |
| 최악 레짐 | 저변동 횡보 (수수료 급감) + 펀딩 역전 | 횡보 whipsaw (연속 손절) |
| 최적 레짐 | 고변동 (방향 무관) | 강한 일방향 추세 |

지난 1년(하락+고변동)은 LP에 최적, 1h EMA에 최악이었다. **강한 상승 추세장이 오면 순위가 바뀔 수 있다** — 방향성 전략은 그런 해에 +50~100%도 가능하고, LP는 비슷한 수익을 유지한다.

## 3. 리스크의 성격 차이 — LP의 낮은 MDD를 액면대로 믿지 말 것

- **EMA 전략의 리스크는 백테스트에 전부 보인다**: 연속 손절, MDD 27.6%가 곧 실전 리스크. 대신 꼬리 리스크는 작다 (거래소 리스크뿐, 킬스위치로 컷 가능).
- **LP의 리스크는 백테스트에 안 보인다**: MDD 0.56%는 모델상 수치일 뿐. 실전에는
  ① 스마트컨트랙트 해킹 (발생 시 -100%), ② 스테이블 디페그, ③ 레인지 이탈 구간 수수료 공백 + 재배치 슬리피지, ④ 펀딩 역전 장기화 — 가 있다. 연속형 지표(Sharpe 21)는 이 꼬리를 반영하지 못하므로 실질 Sharpe는 2~4 수준으로 보는 게 정직하다.
- 요약: **EMA = 자주 아프고 크게 안 죽음 / LP = 안 아프다가 드물게 크게 아플 수 있음**.

## 4. 그 외 객관적 차이

| 항목 | 델타뉴트럴 LP | EMA 매매 |
|---|---|---|
| 자본 용량 | ~$100만 (풀 TVL 제약) | 사실상 무제한 (BTC perp) |
| 자본 효율 | 증거금 묶임 (총자본의 ~14%) | 레버리지로 조절 가능 |
| 운영 복잡도 | 높음 (2개 시스템: 온체인+perp) | 낮음 (perp 단일) |
| 검증 상태 | 1년 백테스트 + DRY_RUN 중 | 백테스트만 (테스트넷 주문 13건) |
| 수익 예측 가능성 | 높음 (캐리 성격) | 낮음 (추세 의존) |

## 5. 결론

1. **위험조정수익 기준, 지난 1년은 LP가 압도적** — 단, 기간이 LP에 유리했고 LP의 꼬리 리스크는 지표 밖에 있다는 두 가지 할인 요인을 반드시 적용할 것. 할인 후에도 보수 기대치(연 12~20%, Sharpe 2+)는 EMA 4h(+7.2%, Sharpe 1.0)보다 우위.
2. **1h EMA 크로스는 그대로 쓰면 안 된다** — 같은 스펙으로 -20.4%. quant 봇 MVP는 4h 기준으로 변경하거나 레짐 필터(변동성/추세 강도) 추가가 필요하다.
3. **둘은 경쟁 관계가 아니라 보완 관계** — 수익 원천이 무상관(캐리 vs 추세 알파)이므로 병행이 합리적. DEFI_YIELD_AGENT_PLAN.md의 코어(LP 70~80%)/위성(방향성 20~30%) 배분과 일치한다.
4. 실무 우선순위: LP 라이브 검증(진행 중) → quant 봇은 4h + 레짐 필터로 백테스트 보강 후 투입.
